Blitzkrieg 3 Patcher

How to run 

• place .exe into your [...\\SteamLibrary\steamapps\common\Blitzkrieg3\ folder

• run Blitzpatch.exe

• select between 'simple' or 'manual' 

-------------------------------------------------------

► Blitzpatch Simple
The 'simple' setting will quickly patch the game so all missions and theaters are unlocked. Additionally, you'll be given a balanced deck of units for every faction. 

► Blitzpatch Manual 
The ‘manual’ setting will give you the ability to unlock or lock missions. Additionally you’ll be able to edit unit experence values, add units to your deck (even units not from your faction), and tweak values (use late war tanks in early war)
